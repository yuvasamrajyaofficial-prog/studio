# "Cosmic" Visual Language

## 1. Color Palette (Deep Space & Divine Glow)

- **Backgrounds:** Deep Obsidian (`#050505`), Midnight Indigo (`#0A0E1A`).
- **Accents (The Glow):**
  - **Sattvic Gold:** `#FFD700` (Wisdom, Purity).
  - **Cosmic Violet:** `#8A2BE2` (Intuition, Connection).
  - **Shanti Teal:** `#20B2AA` (Peace, Healing).
  - **Karmic Amber:** `#FFBF00` (Action, Energy).
- **Gradients:** Use "Nebula" gradients for buttons and cards.

## 2. Typography

- **Headlines:** _Cinzel_ or _Playfair Display_ (Serif, elegant, authoritative).
- **Body:** _Inter_ or _Montserrat_ (Sans-serif, highly readable, modern).
- **Sanskrit:** _Tiro Devanagari_ (Traditional and beautiful).

## 3. Haptics & Sound (The "Vibe")

- **Haptics:** "Soft" haptic pulses when interacting with sacred elements. "Resonant" vibration when a Karma point is earned.
- **Ambient Sound:** Subtle, low-frequency "Om" drone or space-ambient sounds in the background (toggleable).
- **UI Sounds:** Glassy, crystalline chimes for notifications.

## 4. UI Elements

- **Glassmorphism:** Use frosted glass effects for overlays to maintain depth.
- **Micro-animations:** Floating particles, glowing borders, and "breathing" buttons.
- **Icons:** Thin-line, celestial-themed icons (Stars, Mandalas, Lotuses).
